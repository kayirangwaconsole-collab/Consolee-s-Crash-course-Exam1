# How to Enable GitHub Pages for This Project

## Step-by-Step Instructions (Simple)

### **Step 1: Go to Your Repository Settings**
1. Open: https://github.com/kayirangwaconsole-collab/Consolee-s-Crash-course-Exam1
2. Click the **Settings** button (top right of the repository)

### **Step 2: Find Pages Settings**
1. On the left menu, click **Pages** (scroll down to find it)

### **Step 3: Enable GitHub Pages**
1. Under "Source", select **Deploy from a branch**
2. Select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. Click **Save**

### **Step 4: Wait for Deployment**
- GitHub will process this in about 1-2 minutes
- You'll see a green checkmark when it's done
- The site will be published at:
  ```
  https://kayirangwaconsole-collab.github.io/Consolee-s-Crash-course-Exam1/
  ```
